Dampfturbine

Dampf wird in einer gut isolierten, stationär laufenden Turbine von Zustand 1 (p = 2 MPa, T = 500 °C) auf Zustand 2 (p = 10 kPa, x_Dampf = 0.95) entspannt. Die Änderungen in kinetischen und potentiellen Energien können vernachlässigt werden.
  1. Bestimmen Sie die spezifische Turbinenarbeit, Wdot_CV/mdot, in kJ/kg und die spezifische Entropieproduktionsrate, sigmadot_CV/mdot, in kJ/(kg K).
  2. Ist der beschreibene Prozess möglich? Erklären Sie Ihre Antwort.
  3. Zeichnen Sie den Prozess in einem T-s Diagramm
  4. Bestimmen Sie die spezifische Turbinenarbeit in kJ/kg, für den Fall, dass die Turbine den Dampf reversibel auf den gleichen Enddruck entspannt.
  5. Bestimmen Sie den isentropen Wirkungsgrad der Turbine, din in Teil a berechnet wurde.

              = 2465.06 kJ/kg
Massenbilanz
dm_CV/dt = mdot_1 - mdot_2
?Annahme gleiche Massenströme? => dm_CV/dt = 0
=> dm_CV/dt = mdot_1 - mdot_2 = 0
=> mdot_1 = mdot_2 = mdot Energiebilanz
dE_CV/dt = Delta_KE + Delta_PE + mdot_1 h1 - mdot_2 h2 + Wdot_CV + Qdot_CV
  • kinetische und potentielle Energien vernachlässigbar: Delta_KE = Delta_PE = 0
  • stationär: dE_CV/dt = 0
  • gut isoliert, adiabat: Qdot_CV = 0
es folgt
dE_CV/dt = 0 = mdot_1 h1 - mdot_2 h2 + Wdot_CV
Wdot_CV = mdot_2 h2 - mdot_1 h1
mit Massenbilanz
Wdot_CV = mdot (h2 - h1)
Wdot_CV/mdot = h2 - h1 = 2465.06 kJ/kg - 3467.6 kJ/kg = -1002.5 kJ/kg Entropiebilanz
dS_CV/dt = mdot_1 s1 - mdot_2 s2 + Qdot_CV/T + sigmadot_CV
  • stationär: dS_CV/dt = 0
  • gut isoliert, adiabat: Qdot_CV = 0
daraus folgt
dS_CV/dt = mdot (s1 - s2) + sigmadot_CV = 0
=> sigmadot_CV/mdot = s2 - s1 = 7.7752 kJ/kg K - 7.4317 kJ/kg K = 0.3435 kJ/kg K
